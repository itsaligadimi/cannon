# cannon

A Symfony-based REST API with JWT authentication and role-based access control.


## Prerequisites

- **Docker** (and Docker Compose)
- **Composer**

## Setup and Run
```bash
git clone <repository-url>
cd <project-directory>
./docker-setup-and-run.sh
```



    
## Running Tests
```bash
./docker-setup-and-run.sh
```
